## Tools

### Cargo
Install rust ([docs](https://www.rust-lang.org/tools/install))
```shell
cargo install eza
```

### Brew
```shell
brew install neovim
brew install starship
brew install tmux
brew tap homebrew/cask-fonts && brew install --cask font-fira-code-nerd-font
brew install lazygit
brew install ripgrep
```
